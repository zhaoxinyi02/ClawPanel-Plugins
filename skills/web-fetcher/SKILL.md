# Web Fetcher

## 用途
使用本地 Python 脚本抓取网页内容，提取标题和可读文本，适合轻量的页面采集与预处理任务。

## 适用场景
- 需要快速读取网页正文
- 需要在本地做简单抓取验证
- 需要把页面内容交给后续技能继续分析

## 输入
- `url`: 目标网页地址

## 输出
- 页面标题
- 纯文本正文
- 抓取状态或错误信息

## 注意事项
- 仅用于用户明确授权的网页抓取任务
- 目标站点若有访问限制，应遵守对方规则
