# 旅行规划师

## 概述

本技能是一个全面的旅行规划助手，能够记录您的旅行偏好，并为任何目的地生成详细的个性化旅行计划，包括行程安排、预算分解、行李清单和文化指南。

## 工作流程

### 第一步：收集用户旅行信息

请根据用户输入收集用户旅行信息。

**旅行风格与预算：**
- 预算级别：经济、中档、豪华
- 旅行节奏：悠闲、适中、紧凑
- 住宿偏好：青旅、酒店、民宿、度假村
- 出行同伴：独行、情侣、家庭、团体

**兴趣与活动：**
- 观光与地标
- 美食与烹饪体验
- 冒险与户外活动
- 文化与历史
- 海滩与休闲
- 夜生活与娱乐
- 购物
- 自然与野生动物
- 摄影
- 健康与水疗

**饮食与健康：**
- 饮食限制（素食、纯素、过敏）
- 无障碍需求
- 健康注意事项
- 体能水平

**语言与技能：**
- 会说的语言
- 旅行经验水平
- 对冒险的适应程度

**必要信息：**
1. **目的地**：城市/国家
2. **日期**：出发和返回日期（或弹性日期范围）
3. **时长**：天数
4. **预算**：总预算或每日预算
5. **目的**：度假、商务、特殊场合
6. **必看/必做**：特定景点或活动

### 第二步：目的地调研

使用网络搜索收集最新信息：

**必要调研内容：**
1. **最佳旅游时间** - 天气、季节、节庆
2. **安全信息** - 旅行警告、安全区域、常见诈骗
3. **当地交通** - 地铁、公交、出租车、APP
4. **热门景点** - 必看地点及开放时间和价格
5. **美食推荐** - 当地特色、热门餐厅
6. **街区介绍** - 住宿区域、游览区域
7. **一日游选项** - 周边景点
8. **实用信息** - 货币、小费、电源插座、语言

**每项需要调研的内容，都必须执行网络抓取。**

### 第三步：交通调研
#### 交通方式-飞机调研
1. 先通过browser打开携程网址
2. 搜索从出发地到目的地的机票
3. 在搜索结果界面滚动画面，获得所有的机票信息
4. 总结，推荐出行的交通工具
#### 交通方式-火车调研
[SKILL_DIR]是当前技能目录
根据用户出行时间， 
如2026-03-01从杭州到北京， 2026-03-08回来 你需要分别执行以下调用
```
node [SKILL_DIR]/scripts/query.mjs 
杭州 北京 --json -d 2026-03-01
```
```
node [SKILL_DIR]/scripts/query.mjs 
北京 杭州 --json -d 2026-03-08
```
综合火车和飞机返回信息和用户出行信息，推荐出行交通

### 第四步：生成详细旅行计划

创建包含所有组成部分的完整计划：

**A. 逐日行程**

根据用户的节奏偏好和调研结果安排每一天：

```
第1天：抵达 & 哥特区
- 上午（9:00）：抵达巴塞罗那，酒店入住
- 上午（11:00）：哥特区徒步游览
  - 巴塞罗那大教堂
  - 皇家广场
  - 兰布拉大道（简短漫步）
- 下午（14:00）：在 Cal Pep 餐厅午餐（小吃）
- 下午（16:00）：毕加索博物馆
- 傍晚（19:00）：在 El Born 街区用晚餐
- 晚上（21:00）：沿海滨漫步

交通：乘地铁从机场出发（30分钟，€5）
预估费用：€120/人（餐饮、博物馆、交通）
备注：请提前在线购买毕加索博物馆门票
```

每天重复上述结构，确保：
- 地理位置合理分组
- 留有缓冲的合理时间安排
- 活动类型多样化
- 餐饮建议
- 交通细节
- 预估费用
- 预订提示

**B. 预算分解**
按如下格式呈现：

```
总预算：$2,500（7天）
每日平均：$357

明细：
- 住宿：$875（35%）- $125/晚
  * 哥特区精品酒店
  * 含早餐

- 餐饮：$625（25%）- $89/天
  * 早餐：已含
  * 午餐：$25-30/人
  * 晚餐：$40-50/人
  * 零食/饮料：$15/天

- 活动：$625（25%）- $89/天
  * 圣家堂：$35
  * 桂尔公园：$13
  * 毕加索博物馆：$15
  * 美食游览：$95
  * 蒙特塞拉特一日游：$50

- 交通：$250（10%）- $36/天
  * 机场接送：单程$35

```

## 最佳实践

1. **深入调研** - 使用网络搜索获取最新、准确的信息
2. **切合实际** - 不要安排过满，留出休息和随性探索的时间
3. **提前预订** - 热门景点尤其在旺季容易售罄
4. **预留缓冲** - 额外预留10-20%应对意外费用
5. **尊重文化** - 调研并遵循当地习俗
6. **安全第一** - 查阅旅行警告，向大使馆登记
7. **保持灵活** - 天气和情况随时可能变化
